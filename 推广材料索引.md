## 🎯 如何使用这些材料

### 第1步：即时推广（今天就可以做）

#### 微信公众号
1. 复制 `公众号文章.md` 内容
2. 调整格式和配图
3. 替换 GitHub 链接为实际地址
4. 发布文章

#### 朋友圈
1. 从 `朋友圈推广文案.md` 选择合适版本
2. 配合截图或数据图
3. 发布到朋友圈
4. 分享到相关微信群

### 第2步：扩展推广（本周内）

#### 知乎
1. 使用 `公众号文章.md` 或 `精简版.md`
2. 调整为知乎风格（添加目录、优化排版）
3. 添加 #单细胞测序 #生物信息学 话题
4. 发布文章

#### 小红书
1. 使用 `朋友圈推广文案.md` 中的简洁型
2. 制作精美配图（可用 Canva 等工具）
3. 添加相关标签
4. 发布笔记

### 第3步：视频推广（2周内）

#### B站视频
1. 参考 `推广计划.md` 中的视频内容规划
2. 录制：
   - 快速上手（5分钟）
   - 功能演示（10分钟）
   - 实战案例（15分钟）
3. 视频标题参考朋友圈文案
4. 视频简介使用精简版文章

### 第4步：系统推广（长期）

参考 `推广计划.md` 执行：
- 按时间线推广
- 发布系列文章
- 建立用户社区
- 收集反馈改进
- 持续更新内容

## 🎨 配图建议

### 需要制作的图片

#### 1. 项目 Logo
- **尺寸**：1024x1024（可缩放）
- **风格**：简洁、专业、科技感
- **元素**：DNA螺旋 + 数据转换箭头
- **用途**：公众号封面、GitHub README、各平台头图

#### 2. 格式支持示意图
```
[多种格式图标] → [scRNA-DataHub] → [H5AD图标]
   10X MTX         智能识别          统一输出
   DNB C4          一键转换
   CSV/Excel
   ...
```
- **工具**：draw.io、ProcessOn、Figma
- **配色**：蓝色系（科技感）

#### 3. 使用对比图
```
左边（手动代码）：        右边（scRNA-DataHub）：
[复杂代码50行]    vs    [1行命令]
[2小时]          vs    [20分钟]
[❌错误频发]      vs    [✅一次成功]
```

#### 4. 测试结果截图
- 截取实际测试输出
- 标注重点信息
- 添加文字说明

#### 5. 性能数据可视化
- 柱状图：格式数量对比
- 折线图：处理速度对比
- 饼图：格式支持占比

### 制图工具推荐
- **在线工具**：
  - Canva（易用，模板多）
  - 创客贴（中文友好）
  - 稿定设计（专业）
  
- **专业工具**：
  - Figma（免费，功能强）
  - Sketch（Mac专用）
  - Adobe Illustrator（专业）
